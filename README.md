# Predicting Addiction Based on Personal and Social Factors

## Overview
This project uses data analysis and machine learning techniques to assess addiction risk based on personal, social, and behavioral factors.  
The aim is to identify key influencing factors and support early intervention strategies.

---

## Social Objective
- Analyze behavioral and social data  
- Identify risk factors influencing addiction  
- Support data-driven awareness and prevention  
- Generate insights for early intervention  

---

## Key Features
- Data preprocessing and normalization  
- Feature engineering and selection  
- Classification model training and evaluation  
- Performance analysis and result interpretation  

---

## Tools & Technologies
- Python  
- Pandas, NumPy  
- Matplotlib  
- Jupyter Notebook  

---

## Outcomes & Learnings
- Improved understanding of social data analysis  
- Learned classification and evaluation techniques  
- Developed insight-generation and interpretation skills
